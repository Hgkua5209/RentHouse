Objective



The Sports Achievement Tracking System is a web-based platform built on a Django framework with a database backend that aims to serve five main purposes for college use. Firstly, it allows students to track their sports achievements, Such as studentId, LecturerId, sportCode, and session, date, and achievement. Secondly, it provides Managing Student Data, The website will allow Lecturer to manage and monitor Student. Thirdly, Managing Lecturer Data, this website will allow Lecturer to manage their own data and set which lecturer want to monitor which student the data that is inside the lecturer model. Fourthly, Managing Sport Data, including SportCode, SportName, and SportDescription. Finally, it provides a comprehensive record of the college's sporting history, including past achievements and the session wit the date of the sport achivment, which can be used to inform decision-making and inspire future generations of athletes. Through these five purposes, the Sports Achievement Tracking System provides a powerful tool for enhancing the college's sporting culture and promoting the success of its athletes.
